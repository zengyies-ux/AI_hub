# 自媒体图文生成系统 - 项目说明

## 📖 项目简介

**自媒体图文生成系统** 是一个基于AI的自动化内容生成工具，能够从国际热点新闻中提取信息，生成专业的自媒体图文内容，包括热点分析、配图和多平台文案。

### ✨ 核心功能

- **全自动工作流**：从新闻爬取到自媒体内容生成的完整流程
- **多平台新闻源**：新浪、网易、百度、腾讯国际新闻
- **AI深度分析**：智能识别热点话题，生成深度分析报告
- **文生图提示词**：为每个热点生成专业的AI绘画提示词
- **多平台内容**：支持小红书、抖音、微信公众号等平台格式
- **3:4图片比例**：生成适合社交媒体的竖版图片
- **智能图片下载**：自动下载并按话题智能命名
- **无内容限制**：完整保留所有信息，无token消耗限制

## 🔧 技术架构

### 工作流程

1. **数据采集**：从四大新闻平台爬取国际热点新闻
2. **数据处理**：合并相似话题，提取核心信息
3. **AI分析**：生成深度分析报告，搜索补充信息
4. **提示词生成**：为每个热点生成专业的文生图提示词
5. **图片生成**：调用Coze API生成3:4比例的配图
6. **内容创建**：生成适合不同平台的自媒体文案
7. **图片下载**：自动下载图片并按话题智能命名

### 核心技术

- **网络爬虫**：使用Requests和BeautifulSoup爬取新闻
- **自然语言处理**：使用Jieba进行中文分词和关键词提取
- **AI接口**：使用DeepSeek API生成提示词，Coze API生成图片
- **数据存储**：结构化存储为Markdown和JSON格式
- **文件管理**：智能文件命名和目录组织

## 🔄 常见问题

**Q: 运行时出现SSL错误怎么办？**
A: 运行 `pip install urllib3==1.26.18` 或在代码中禁用SSL警告

**Q: 爬取新闻失败怎么办？**
A: 检查网络连接，稍后重试，或更换User-Agent

**Q: 生成的图片质量不佳怎么办？**
A: 可以在 `提示词生成器.py` 中调整提示词生成参数

**Q: 自媒体内容格式不满意怎么办？**
A: 可以修改 `主控制器.py` 中的内容生成模板

**Q: 图片下载失败怎么办？**
A: 检查网络连接，或手动下载图片链接中的图片
