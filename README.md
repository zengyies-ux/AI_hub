#  自媒体图文生成系统

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**基于 AI 的自动化自媒体内容生成工具** —— 从国际热点新闻爬取到多平台图文内容一键生成

---

## ✨ 核心功能

| 功能 | 描述 |
|------|------|
| 📰 **新闻爬取** | 自动爬取新浪、网易、百度、腾讯国际新闻 |
| 🤖 **AI 深度分析** | 使用 DeepSeek AI 生成专业热点分析报告 |
| 🎨 **智能提示词** | 为每个热点生成专业的 AI 绘画提示词 |
| 🖼️ **AI 配图生成** | 调用 Coze API 生成 3:4 竖版配图 |
| 📱 **多平台内容** | 自动生成小红书、抖音、公众号文案 |
| 💾 **图片下载** | 自动下载图片并按话题智能命名 |

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/zengyies-ux/AI_hub.git
cd AI_hub
```

### 2. 安装依赖

```bash
python3 -m pip install -r requirements.txt --user
```

### 3. 配置 API 密钥

```bash
cp config.example.json config.json
```

配置文件格式：

```json
{
    "deepseek": {
        "base_url": "https://api.deepseek.com/v1",
        "api_key": "你的DeepSeek API密钥"
    },
    "coze": {
        "api_url": "你的Coze图片生成API地址"
    }
}
```

> **获取 API 密钥**：
> - DeepSeek: https://platform.deepseek.com/
> - Coze: https://www.coze.cn/

### 4. 一键启动

```bash
python3 启动.py
```

- 自媒体内容生成

获取地址：https://platform.deepseek.com/

- AI 图片生成（3:4 竖版）

获取地址：https://www.coze.cn/

## 📝 使用建议

1. **定期运行**：建议每天运行一次，获取最新热点
2. **人工审核**：生成的内容建议人工审核后发布
3. **API 用量**：注意监控 API 调用量和费用
4. **存储管理**：定期清理 `输出文件/` 目录下的历史文件

---

## ⚠️ 注意事项

- 请勿将 `config.json` 提交到公开仓库
- 爬取新闻时会有随机延迟，避免被反爬
- 生成的内容仅供参考，请以官方信息为准
- API 调用会产生费用，请注意用量
